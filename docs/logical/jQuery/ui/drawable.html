protected jQuery.ui.drawable _get_drag_left_position (Event event)
Get the new left position for the proxy based on the current position and the drag event. If the mouse is left of the initial click point, we adjust left along with the mouse so that a left drag is possible. We use the grid settings to snap into place
protected jQuery.ui.drawable _get_drag_top_position (Event event)
Get the new top position for the proxy based on the current position and the drag event. If the mouse is above the initial click point, we adjust top upwards so that we can drag further up. If it is below, we keep it the same. In either case, we use the grid settings to snap into place
